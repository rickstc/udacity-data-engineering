## Quickstart

### Install the Dependencies

To run this application locally, the dependencies must be installed (preferably in a virtual environment).

#### Using Pipenv (preferred)

```
pipenv install
pipenv shell
```

#### Using Pip

```
pip install -r requirements.txt
```
